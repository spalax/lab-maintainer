Welcome to Lab-Maintainer.

It is a specific oriented project for company's Laboratory built for simplify detectors enrollment and attaching them to the Power Max systems. Lab-Maintainer consists of reusable Pintura modules and Perstore it can be used on any JSGI compliant JavaScript platform, but is tested on NodeJS.

Setup Lab-Maintainer

$  git clone https://github.com/Spalax/lab-maintainer.git lab-maintainer
$  cd lab-maintainer
$  npm install
$  git submodules init
$  git submodule update
